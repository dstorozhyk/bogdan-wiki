# Пошук Локальних Спеціалістів в Україні

Система для пошуку та управління контактами сантехніків, електриків, монтажників та інших майстрів по Україні.

## Використання

```bash
python3 ~/.hermes/skills/ukraine-specialists-finder/scripts/find_specialist.py <тип> <місто>
```

### Приклади

```bash
# Електрики Львова (відсортовані по рейтингу)
python3 ~/.hermes/skills/ukraine-specialists-finder/scripts/find_specialist.py электрики lviv

# Монтажники кондиціонерів Львова
python3 ~/.hermes/skills/ukraine-specialists-finder/scripts/find_specialist.py монтажники_кондиціонерів lviv

# Всі спеціалісти у місті
python3 ~/.hermes/skills/ukraine-specialists-finder/scripts/find_specialist.py --all lviv
```

## Как додавати нових спеціалістів

1. Знайди спеціаліста на OLX, Kabanchik, Telegram або Facebook
2. Отримай інформацію: ім'я, телефон, рейтинг, спеціальність, вартість
3. Редагуй файл `~/.hermes/skills/ukraine-specialists-finder/references/<city>_specialists.json`
4. Додай новий запис із структурою:

```json
{
  "id": "elec_004",
  "name": "Ім'я Прізвище",
  "phone": "+38 (0XX) XXX-XX-XX",
  "rating": 4.5,
  "speciality": "Опис послуг",
  "source": "OLX / Telegram / Facebook / Viber",
  "url": "https://посилання",
  "response_time_hours": 2,
  "price_range": "200-500 грн/час"
}
```
